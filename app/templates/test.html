基本課程大綱

初始的每週課程結構

Python 程式設計基礎

課程概述

這門 16 週的課程為計算機科學一年級學生介紹 Python 程式設計基礎。

每週課程安排

第 1 週：程式設計簡介

  • 學習目標：
    • 理解什麼是程式設計
    • 設置 Python 開發環境
    • 撰寫你的第一個 Python 程式
  • 涵蓋主題：
    • Python 的歷史
    • 安裝 Python 和 IDE
    • 基本語法和結構
    • 列印語句和註解
  • 活動：
    • Hello World 程式
    • 環境設置實驗
    • 基本輸入/輸出練習

第 2 週：變數和資料型別

  • 學習目標：
    • 理解 Python 中的不同資料型別
    • 學習變數命名約定
    • 練習型別轉換
  • 涵蓋主題：
    • 數字（int, float）
    • 字串和字串操作
    • 布林值
    • 變數賦值和命名
增強課程大綱

網路增強的課程大綱，包含額外資源

Python 程式設計基礎 - 增強版

課程概述

這門綜合性的 16 週課程為計算機科學一年級學生介紹 Python 程式設計基礎，並結合業界相關範例和實作專案。

每週課程安排

第 1 週：程式設計簡介與開發環境

  • 學習目標：
    • 理解程式設計及其應用
    • 設置專業的 Python 開發環境
    • 撰寫並執行你的第一個 Python 程式
    • 了解軟體開發生命週期
  • 涵蓋主題：
    • Python 的歷史和哲學
    • 安裝 Python、PyCharm/VSCode 和 Git
    • 命令列基礎
    • 虛擬環境和套件管理
    • 程式碼版本控制概念
  • 資源：
    • Python.org 官方教學
    • Real Python - Python 基礎
    • Automate the Boring Stuff
  • 評估：
    • 環境設置檢查表
    • 簡單程式執行

第 2 週：變數、資料型別與記憶體管理

  • 學習目標：
    • 掌握 Python 的內建資料型別
    • 理解記憶體分配和變數範圍
    • 練習專業命名約定
    • 實作型別檢查和驗證

專案導向學習

  • 期中專案：個人財務計算器
  • 期末專案：文字冒險遊戲
  • 持續評估：每週程式設計挑戰
