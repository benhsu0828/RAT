User Manual

Workflow
How to Use
Navigation
  • Home – Returns to the homepage.
  • Course Outline Generation – Opens the course outline generation page.
  • System Info – Opens the system information page.

The top-right corner displays the current AI model in use (e.g., "OpenAI").

Homepage
  • Click Course Outline Generation to create a course outline.
  • Click System Info to view system information.
Course Outline Generation Page

This page allows you to generate a course outline in two ways:

  1. Manual Input
    • Course Title (required)
    • Target Audience (required)
    • Course Duration (required) – Total number of weeks.
    • Class Hours (required) – Number of hours per week.
    • Core Requirements (optional) – Key topics or skills to include.
    • Additional Requirements (optional) – Extra notes or specific needs.
  2. Upload Course Document
    • Click Upload Course Document to upload a PDF file.
    • The system will automatically extract content and fill in the form fields.
Generating the Outline
  1. After completing the form or uploading a document, click Generate Course Outline.
  2. Wait approximately 3 minutes for the system to process the request.
  3. Scroll to the bottom of the page to find:
    • Copy – Copies the generated outline to your clipboard.
    • Download File – Downloads the outline as a file.
System Information Page

Displays system information for reference. This page is read-only; no changes can be made.
